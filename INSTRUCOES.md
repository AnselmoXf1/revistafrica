## 🚀 Como Executar o Site

1. Abra o terminal na pasta REID-main
2. Execute os comandos:

```bash
npm install
npm run dev
```

3. O site abrirá no navegador

## 👨‍💼 Como Publicar Artigos (Admin)

1. Acesse o site
2. Clique em "Área Admin" (canto superior direito)
3. Faça login com suas credenciais
4. Clique em "Publicar" no menu
5. Preencha os dados do artigo e faça upload do PDF
6. O artigo aparecerá automaticamente na página inicial

## 📧 Como Funciona a Submissão de Artigos

1. Usuários clicam em "Submeta seu Artigo Aqui"
2. Abre automaticamente o email revistaafricaa@gmail.com
3. Eles enviam o artigo por email
4. Você recebe o email
5. Após revisão, você publica o artigo pelo painel admin
6. O artigo aparece na página inicial em PDF
